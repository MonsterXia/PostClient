# 邮差好友公会服务器规定(V2.6)

[TOC]

## 一、总则

### 1.简介

邮差好友公会玩家服务器规定参考了各个国服的服规并进行了一定程度的补充与完善。会根据实际情况实时更新，有发现服务器规定有漏洞需要补充的请联系OP和群管理员，谢谢各位。

### 2.邮差好友公会服务器规定的任务、基本原则和适用范围

- 立规目的:为了维护服务器内玩家正常游玩的权利，结合实际处置具体经验及实际情况，制定本规定。
- 任务:邮差好友公会玩家服务器规定的任务，是用规定同一切破坏维护服务器内玩家正常游玩的权利的行为作斗争，以保证维护服务器内玩家正常游玩的权利。
- 处罚规定:规定明文规定为可以处罚的，依照规定处罚；规定没有明文规定为可以处罚的，不得滥用处罚。
- 规定面前人人平等:对任何人违反规定，在适用规定上一律平等。不允许任何人有超越规定的特权。
- 行为处罚相适应:处罚的轻重，应当与违反服务器规定的行为和承担的处罚相适应。
- 管辖权：凡在邮差好友公会服务器内违反服务器规定的，除规定有特别情况的以外，都适用本规定。
- 溯及力：本规定施行以前的行为，如果当时的规定不应该处罚的，适用当时的规定；如果当时的规定认为是应当处罚的，依照[《一、总则;4.追诉时效期限》](#4追诉时效期限)的规定应当追诉的，按照当时的规定追究处罚，但是如果本规定认为是不应该处罚的或者处罚较轻的，适用本规定。

### 3.更新

邮差好友公会服务器规定以最新版为标准，自新版发布公示后，旧版即废止。

### 4.追诉时效期限

违反服务器规定经过下列期限不再追诉：

- 处罚不涉及封禁的，当局接受过处罚或者当局未再次进入对局。
- 处罚涉及封禁的，经过应当封禁的时间后再次进入对局。

追诉期限的延长：

- 在OP受理以后，通过一些方式逃避侦查或者处罚的，不受追诉期限的限制。
- 被害人在追诉期限内提出控告，OP应当受理而未受理的，不受追诉期限的限制。

### 5.自首和立功

- 自首：违反规定以后自动投案，如实供述自己的违反规定行为的，是自首。对于自首者，可以从轻或者减轻处罚。其中，违反规定行为较轻的，可以免除处罚。处罚过程中如实供述服务器管理人员还未掌握的本人其他违反规定行为的，以自首论。
- 立功：违反规定人员有揭发他人违反规定，查证属实的，或者提供重要线索，从而得以在处置其他案件中存在立功表现的，可以从轻或者减轻处罚；有重大立功表现的，可以减轻或者免除处罚。其他游戏玩家在处置案件中提供关键线索的，可以获得处罚天数相当的预留位奖励。

### 6.数罪并罚

- 对于违反服务器多个规定的行为人，除判处永久封禁以外，应当在总和处罚以下、数个处罚中最高处罚以上，酌情决定。
- 判决宣告后发现漏处罚的并罚：判决宣告以后，处罚执行完毕以前，发现被处罚的行为人在判决宣告以前还有其他违反服务器规定没有判决的，应当对新发现的违反服务器规定的行为出判决，把前后两个判决所判处的处罚，依照[《一、总则;6.数罪并罚》](#6数罪并罚)中首条的规定，决定执行的处罚。已经执行的处罚，应当计算在新判决决定的处罚以内。
- 判决宣告后又违反服务器规定的并罚：判决宣告以后，处罚执行完毕以前，被处罚的行为人又违反服务器规定的，应当对新违反服务器规定的行为作出判决，把前处罚没有执行的部分和后处罚，依照[《一、总则;6.数罪并罚》](#6数罪并罚)中首条的规定，决定执行的处罚。

## 二、通用规则 

### 1. 关于击杀队友(TK:TeamKill)

- 误杀队友在公屏按【J】输入sorry/sor/sry等。即使用ALL频道道歉。
- TK队友后未道歉会在一分钟内被Kick(踢出服务器)。
- 恶意TK、TK未道歉并逃逸者将依据情况被Ban3天至永Ban不等。
- 对于造成本阵营损失票数损失的不正当行为，在经其他玩家举报后，OP可以依据以下情况对被举报玩家进行处罚:
  - [ ] 对于直升机，为保证同阵营玩家的游戏体验，经玩家举报，若在较短时间内造成多架直升机非正常坠毁(四次直升机刷新时间内致使三架或以上直升机损毁)，直升机驾驶员会被踢出服务器以保证本阵营其他玩家对局内对直升机的正常使用权。
  - [ ] 对于载具、ATGM，经玩家举报，存在向友军单位开火的行为，如果继续向友军开火，武器操作员将会被Kick(踢出服务器)以减少本阵营其他玩家对局内受到来自友军的火力。
  - [ ] 对于步兵单位，经玩家举报，且被举报的行为造成了友方阵营10tickets以上的损失，被举报的行为人将被Kick(踢出服务器)。
  - [ ] 若玩家多次被依据此条Kick(踢出服务器)，将依据情况被Ban3天至永Ban不等。

### 2.恶意拆除FOB

- 禁止恶意拆除本方FOB，初犯将Ban7天，第2次违反者将永Ban。
- 恶意拆除本方FOB并逃逸者，初犯即永Ban。
- 若是小队长要求队员或亲自恶意拆己方FOB，队长初犯即Ban7天，第3次犯永Ban。若逃逸则初犯即永Ban，实际执行的小队员踢出服务器。

### 3.辱骂或口臭 

- 当值OP有权对公屏言论进行合理管辖。
- 嘲讽其他玩家的行为人无论先后经警告无效后Kick(踢出服务器)。
- 辱骂他人的玩家无论先后都Kick(踢出服务器)。
- 对于以下几种加重型，将会按照《嘲讽和辱骂封禁时间细则》进行加重处理，对行为人进行相应的踢出服务器/封禁：


- [ ] 经Kick(踢出服务器)/Ban后，仍旧实行嘲讽、辱骂行为的。
- [ ] 带头或以某战队名义辱骂、嘲讽、顶撞玩家、服务器及管理人员的。

### 4.比赛

- 各个玩家请积极比赛。
- 若队长存在消极带队、带队郊游的情况，通过弹劾程序(详见[《七、弹劾》](#七弹劾))后将被踢出服务器。第3次违反者将视情节严重性处以Ban7天至永Ban不等。

### 5.玩家ID、聊天频道、队名规范 

- 禁止玩家ID、小队名称、任意聊天频道或语音频道、队伍名称侮辱其他玩家，违反国家法律法规、道德，存在种族歧视、涉及政治等行为。
- 禁止在公屏讨论政治内容（包括但不限于涉及邪教、国家分裂、政治）。
- 处罚方法：


- [ ] 对于玩家名称违规，无需警告直接踢出服务器并声明修改后再进入服务器。
- [ ] 对于小队名称名称违规，经OP警告2次以上无效后解散队伍。
- [ ] 任何试图宣扬邪教、国家分裂的ID以及言论将不经警告直接永Ban。

### 6.直播、窥屏、报点 

- 禁止未经中国邮差好友公会书面授权的直播行为，一经发现，OP警告3次关闭直播无效后便可踢出服务器。踢出3次仍然违反的，将永Ban。
- 经过邮差好友公会书面授权直播游戏的玩家请将延迟调至10分钟以上或增加地图遮挡。若违反，将依据以下进行处理：


- [ ] 第一次劝告无效后将踢出服务器。
- [ ] 踢出3次仍然违反的，将取消授权且 Ban7天。

书面授权将放于邮差公会玩家群公示。

- 禁止任何玩家窥屏、跳边报点、利用Bug报点行为。一经查证，将视情节，Ban7天至永Ban不等。
- 禁止向对方阵营透露本阵营的任何信息。将视所透露的信息重要程度以警告/踢出/封禁处理。

### 7.广告

- 禁止在本服务器里发布任何招募其他战队或其他任何形式的广告。初犯第一次将被踢出，第3次违反的将视情节严重程度Ban7天至永Ban不等。

### 8.锁队

- 步兵队禁止单人锁队（载具队锁队规则详见[《五、载具;1.载具队锁队、人数限制》](#1载具队锁队人数限制)），经3次警告无效后，队长将被移出小队/踢出服务器。
- 陶队、迫击炮队等为保证其他队伍游戏体验，建议专队锁队专用，但非强制性规则。 但若因此被其他队长投诉，参照本服规则[《二、通用规则； 4.比赛》](#4比赛)。
- 除OP、特定地图的侦察车、陶车、短号车、补给队外，禁止单人锁队。违反者处罚效果同[《二、通用规则；8. 锁队》](#8锁队)。

### 9.麦克风、队长装 

- 队长必须着队长装且拥有麦克风。
- 初犯者经警告3次无效后将被踢出，第3次违反将Ban7天。如遇上队长掉线等特殊情况，请及时向OP说明情况。

### 10.建队甩锅 

- 禁止建队甩锅，将视情节kick/Ban，首次甩锅上限3天。多次甩锅将Ban7天。
- 甩锅逃逸初犯Ban7天，第3次违反者将视情形Ban1至3个月。
- 如因服务器内玩家要求OP从而脱离队伍执法不视作甩锅，有特殊情况请及时向OP说明。

### 11.爆炸致使载具不刷新BUG

- 对IED放置数量不做限制要求。
- 当IED、地雷等爆炸物致使bug发生后，暂时禁用造成BUG的装备，在双方中任一方工兵使用c4炸掉BUG载具残骸，使得载具进入正常可刷新状态时，此装备方可重新启用。
- 禁用以及解禁以op黄字为准。
- 当IED、地雷等爆炸物致使bug发生的爆炸物被禁用后仍旧使用的，以踢出处理，多次踢出后仍旧使用的Ban3天。

### 12.投降

- 当存在指挥官时，由指挥官发起投降。当无指挥官时，由各队长发起投降。
- 在投降方发起投降后由op主持，投降一方阵营至少2/3成员明确同意投降则视为通过。
- 投降计票的开始以及结束以op黄字为准，时长为3分钟。
- 当投降票数未达到2/3时，进入5min的冷却时间。

### 13.补偿

若存在误处理情况，请通过邮差好友公会玩家群向相应负责人提起申请补偿。

- 误处理但未上升到将踢/封，补偿3天预留位。
- 误踢将补偿7天预留位。
- 误封将补偿1个月预留位。

当在对局中受到OP的嘲讽/辱骂，请保留证据并通过邮差好友公会玩家群向相应负责人提起诉讼，若确有其事，将对涉事人员进行处罚，并对相应玩家补偿2个月预留位。

## 三、压家规定

### 1.压家范围、保护

服务器现行规则无任何压家保护。

## 四、FOB 放置 

### 1.Bug

禁止利用任何与FOB有关的Bug。

### 2.FOB存在形式

任何FOB必须至少存在一种能够正常拆除的方式。

### 3.处罚方式

- 初犯者将被踢出服务器，违反第3次及以上者Ban3至7天。
- 若已经放置无法拆除的FOB，该FOB将不可继续使用：

- [ ] 不可卸载、使用该FOB的建材、 弹药。
- [ ] 若已有兵站，不可使用该兵站复活，且需在OP监督下将兵站、固定武器等（陶、 机枪碉堡等）拆除。

## 五、载具 

### 1.载具队锁队、人数限制

- 载具队不限制锁队。允许步坦、步载协同队。
- 但若经其他队长弹劾浪费载具（需要2/3队长投票通过）或并不能有效实施步坦、步载协同，队长将被处罚。
- 所有载具队开局需清晰标注队名。直升机小队若发现恶意摔直升机的行为，将依据[《二、通用规则；1. 关于击杀队友(TK:TeamKill)》](#1-关于击杀队友tkteamkill)和[《五、载具;3.直升机》](#3-直升机)进行处罚。
- 所有需要驾驶成员装驾驶的载具队至少2人,其载具禁止单载。
- 关于侦察车单载条件：


- [ ] 若侦察车是最高武装力载具，此地图情况下禁止单载。
- [ ] 若侦察车不是最高武装力载具，此地图情况下允许单载。

- 若驾驶员或炮手死亡而出现单载情况，请尽快离开交战区并尽快将载具开至合适地点接上死亡的驾驶员或炮手（履带被打断等情况例外）。

### 2.载具建队

- 载具队在开局倒计时阶段将按照建队先后顺序、专项名称分配载具。


- [ ] 当目前名称能唯一指定到当前对局中存在的载具时，其精确等级视为最高，若在先后顺序中处于能获得载具的情况下，具有对此载具的优先使用权。
- [ ] 当目前名称不能唯一指定到当前对局中存在的载具时，但能涵盖某几种载具时，其精确等级视为次之，若在先后顺序中处于能获得载具的情况下，具有对此类载具中的某一个体具有优先使用权，其个体为其他队根据上一前提获得相应载具后此类别中剩余的个体。
- [ ] 即便当一个小队名称具有最高的精确等级时，若按照先后顺序不能获得载具，此名称无法获得相应载具的优先使用。
- [x] 详情判例参见[《§附录》](#附录)。
- 开局后，除延时刷新载具有首次使用保护外，将实行载具先基地认证先得原则或由小队之间自行协商。
- 若对某队使用载具存在滥用的异议，请根据[《七、弹劾》](#七弹劾)进行弹劾，结果由本阵营玩家决定归属权。

### 3. 直升机

- 不限制神风、手术刀行为。
- 恶意摔直升机的将Ban1天，摔机逃逸或第3次违反者Ban7天至永Ban（多局累计）。
- 若本阵营其他队长投诉浪费载具或滥用载具，将依据弹劾结果或[《二、通用规则；1. 关于击杀队友(TK:TeamKill)》](#1-关于击杀队友tkteamkill)进行处罚。

### 4.处罚方式

- 所有初犯视情况将被警告/Kick(踢出服务器)，第3次违反者将视情节严重性Ban7天至永Ban不等。

### 5.暂行认定

参考实际划分以及世外相关划分，对于容易引发争议的定义做出以下暂行分类：

- 侦察车定义：除武装运输载具、补给载具外所有不需要载具成员装即可驾驶的武装载具。
- 对于郊狼(Coyote/LAVII RV),虽作为LAV25衍生型,但后期世外将其独立为Coyote，需要以Coyote/郊狼/土狼/Coy等明确指代才能认定唯一，不可用LAV，对于加军阵营，LAV 6.0与LAV具有等效性。
- 穿甲弹、高爆弹攻击FV4034丛林涂装的泥浆书包、高爆弹攻击LAV25与Coyote的备用轮胎视作使用BUG。


##  六、作弊

### 1.严禁作弊

通过第三方作弊软件、BUG等获得优势的行为视为作弊。

- 使用第三方作弊软件一经发现，将录像取证并不经警告永Ban，并提交国服联Ban，同时将证据上交世外（OWI）官方。
- 禁止利用游戏内Bug，根据性质程度将处以踢出至封禁不等的惩罚，包括但不限于：

- [ ] 利用Bug装卸无限补给卡。
- [ ] 卡进建筑物里、地面、墙体、水里。

### 2.等效作弊行为界定

- 使用鼠标宏等行为视作为作弊行为。

### 3.疑似行为处理

对疑似但无确凿证据行为，服务器管理将向世外官方通报疑似行为，由官方接手进一步处理。

## 七、弹劾

### 1.目的以及方式

- 目的：为保证本阵营全体玩家的权利。
- 方式：玩家可以通过自己的小队队长向OP提出弹劾(附带弹劾对象、理由)，实现对特定玩家的弹劾。

### 2.规则

- 弹劾由队长发起，OP作为公证人主持该阵营队长投票。结果取决于该阵营所有小队队长投票结果，投票选项分为赞成/弃权/反对。
- 弹劾发起时当前阵营队长的票为有效票。
- 当赞成票票数等于或超过有效票的2/3时，弹劾通过，否则进入冷却阶段。
- 弹劾进行时间为2分钟，以OP黄字发起弹劾时间起计算。从OP发起到弹劾，2分钟后根据投票结果处理。
- 单次弹劾时间冷却为3分钟，3分钟内不可再次对同一人/队发起弹劾。

### 3.结果

- 弹劾未通过进入冷却，待冷却时间结束后可再次发起。
- 弹劾通过后OP将根据弹劾结果对相应玩家进行移出小队/Kick(踢出服务器)/Ban(封禁)。

## 八、暖服 

### 1.暖服阶段定义

服务器人数小于60且有OP主持时视作暖服阶段，暖服限制将于人数到达60人时自动解除。

### 2.暖服规定

为保证暖服阶段玩家游戏体验，兼顾暖服效率，做出以下规定：

## 1.载具归属判例说明

a.

```
当局存在:
    I. BRDM
    II.Tigr-M RWS Kord
依照建队顺序存在:
    1.ZCC
    2.ZCCRWS
    3.BRDM
```

ZCC包含I、II，因此1队可以获得I、II中的某一辆载具。

2队ZCC明确指代II,且II未分配，因此2队可以获得II。

在2队获得II的同时，因为1队可获得的两辆载具中II被分配，只剩I，因此I自动被分配给1队。

至此当局载具完成分配。

3队即便名称指代了某载具，但因尚无载具可分配，不具有优先权利。

b.

```当局存在:
    I. ZBL08
    II.ZBL08 AFT07C
依照建队顺序存在:
    1.0808
    2.0808
    3.0808 HJ
```

0808指代08车族，在当局中指代ZBL08系列的两辆载具I、II，因此1、2两队都可以获得I、II中的某一辆载具。

因只存在I、II两辆08系列载具，因此1、2两队都获得一辆08，具体II的归属根据建队顺序优先选择。

至此当局载具完成分配。

3队即便名称指代了某载具，但因尚无载具可分配，不具有优先权利。
